Creating a Player

Creating a player is simple and is required to play your game.

  1. First, open the map you would like your player to be on.
  2. Make sure Event View and the cursor tool is checked.
  3. Right click and click .
  4. Double click the player event and a new window should appear.
  5. Add a new party member (a hero) by clicking the green PLUS button in the Party List group box.
  6. Select a hero and press OK.

To learn more about customazing the player, check out tutorials for specific genres such as the RPG tutorials.
