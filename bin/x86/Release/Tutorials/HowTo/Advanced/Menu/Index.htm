Menu

Menus in video games are used for several purposes such as a Title menu, Options menu, or Heads-up Displays.

For this tutorial, we will make a Title menu and explore the basics of the menu maker and how it works. For more advanced tutorials on the Menu maker, see different Menu tutorials.

First, make sure you have imported and added your menu skin. If you do not know how, read the Skin tutorial. You don’t need a skin but it will definitely look better. You’ll also need a Font.

Open the Menu editor . You’ll also need and . Setup the windows anyway you like.

Add a new menu and rename it to “Title Menu.” Go to the Project Settings and select your initial scene as Menu and select the “Title Menu.” Press OK and close it.

Menu Property explorer contains settings for the menu and menu parts. Click on the blank space where the menu parts will be and the menu property should contain settings for the menu. Make the “Default Skin” the skin you would like to use. Although you can have different skins for different menu parts, this will auto-set the skin for your newly added menu parts and save you time.

Menu Parts explorer contains menu parts from Windows to Dynamic Bars to Lists.  For this tutorial, we’ll use just the Static > List part.

Drag and drop the  part from the menu parts explorer to the menu editor. Resize it and reposition it to where you like.

While the Window is selected, open the property explorer and open “Options.” Add 3 new options and select the first.

Scroll down to see the following settings.

This will allow you to rename and pick the font of the option. Each option can have its own Font, Style, Text, and Color. Make sure to pick a color that is visible on the skin. Pick the Font and Style you would like to use. For Text, the first option write “Start Game”, the next “Continue”, and for the third “Exit Game”.

Now that we have our options, we’ll need a way to know which option is selected. Add a new variable and rename it “Title Index.” While the list is selected, fine the Variable property.

Set it to “Title Index”.

Events are not only used by the map but also by the menu. For Menus, you have pre-set event conditions such as OnConfirm, OnCancel, OnShow and etc. They are all listed under the Event category of the properties of a menu part. For this tutorial, we will use OnConfirm and OnSelectedIndex.

Open the OnSelectedIndex event and add a new “Play Audio” command. Select an index change sound effect and Channel 3, the channel that should be used only for general sound effects.

Press OK and open OnConfirm event.

This is the basic idea of the menu maker. First, you design the menu and then use Events to command or “program” them.

Now, think of a simple menu you would like to do and try it out.  There are many menu parts and properties, so drag and drop each to see their purpose. You can also check out the templates to see some of the ways they can be used.
