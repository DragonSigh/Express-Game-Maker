Animations

Animations are grouped graphics for your characters and other objects. Animations are used by Events, Projectiles, Heroes and Enemies.

Open the Animation editor by clicking the Animations button in the menu.

If your game uses sprite sheets (recommended), it is just a few click to create an animation.

  1. Add a new animation by clicking the green PLUS button in the panel "Animations".

  2. Add a new action (Actions) and right-click and rename it to "Walk".
  3. Click the "From Sprite Sheet" button to the right.

The following should appear:

Display time is amount of time each frame should be displayed.

Directions are row based and since not all sprite sheets are in the same order of "Up/Down/ect.", we allow you to re-order the directions.

Basically, if "Up" is in the second row of your sprite sheet, then move "Up" down once. Test it out by importing a walking character from the RP.

If your graphics are not sprite sheets, you'll have to manually add frames to each direction of each action. We are working on making this easier…

Once you create a frame, drag and drop your material from the material explorer. Repeat this process for other frames to get the desired effect. You can have more than one image in a frame.
