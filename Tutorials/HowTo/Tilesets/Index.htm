Tilesets

Tilesets are used by Maps to easily create your enviroments. A tileset is a set of re-usable tiles with default information stored, such as collision and physics settings.

To go on with this tutorial, you'll need to import a tileset to the Materials explorer. If you don't know how, check out the "Importing Materials" tutorial.

To create your tileset, open the Tileset editor .

  1. Add a new tileset by clicking the green PLUS sign.
  2. Drag and drop a tileset image from the Materials explorer to the emptry gray area.
  3. Your tileset image should appear with the default grid size.

One of the most important aspects of tilesets is re-usable information such as collision. To add collision to your tileset:

  1. Click .
  2. You'll see the following tools:

    1. Enables adding of a single node. Best for creating a shape from scratch.
    2. Enables adding of a rectangle. Best for rectangle collisions
    3. Enables adding of a circle. Best for circular collision.
    4. Enables outlining a tile with collision nodes. Best for irregular shaped collisions.
  3. Check the Rectangle tool and click on a tile. A rectangle should outline the tile. This rectangle is made of collision nodes.
  4. Alternatively you can click and drag on the tile to create a custom rectangle.
  5. Make sure to uncheck the Rectangle tool if you want to modify any collision nodes.
  6. To add more nodes to an existing shape, hold [SHIFT] and click and drag on a node.

You can modify a tile(s) default properties such as animation, physics, height (useful for projectiles), and tag at the tab "Tile" to the right.

In order to use a tileset on a map, select it from the Tiles explorer's (View > Tiles) tileset list. To learn about maps so you can use your tileset, check out the next tutorial.
