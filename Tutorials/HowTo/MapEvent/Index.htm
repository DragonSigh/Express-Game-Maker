Map Events

An event is a “point and click” programmable object that activates and triggers at given conditions and performs actions you set it to do. Events can be used to create NPC characters, objects such a doors and chests, and enemies.

In order to understand this tutorial, make sure to have read the previous tutorials.

To create an event on the map, open a map.

Check the button. Double click on the map and a new window should appear.

The event is now created but it needs some life to it. For this tutorial, we create an NPC character that randomly walks around in just a few steps.

First, if you are not using a template, make a new animation of a sprite walking. It only requires one action (name it "Walk") and 4-directions. Make sure to add its collision information.

  1. Double click the gray box at top-left side of the event.
  2. Choose a suitable animation for walking.
  3. Select the default direction and click OK.
  4. Next, we need to tell it to move. Click Program Dynamics.
  5. Click Move Random and press OK.
  6. Check Repeat and Ignore Impassible and press OK.
  7. Close the event dialog and you are done!

The orange circle with the number states the layer the event is on. In this case, this event was on layer 0. You can select an Event only if you are on the layer it is on.

Do not delete this event as you will be able to test it in just a few tutorials later.
