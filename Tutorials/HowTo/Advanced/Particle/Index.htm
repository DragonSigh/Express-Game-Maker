Particle System

Particles are usually used for fluid animations such as fire, smoke, explosions, magic, blood and etc. What makes our particle system great is multiple emitters and robust settings.  

Note: This is an advanced feature.

In order to use the particle system, you’ll need some graphics, mainly “001-Explosion” and “002-Smoke” for this tutorial. You can find the images in either in RPG template, shooter template, or the RPG package.

Open the Particle editor . Add a new particle system, call this Fire.

Our fire will have two components, flames and smoke.

Add a new particle emitter and call this Flames.

Click Select Images and select “001-Explosion”. You can select more than one image by holding down the shift or control keys, but for now, we’ll select just one. Click OK and you will see the image in the Particle Viewer.

As the image is on a black background, we need to change the blending options of the image, from the default Alpha Blend to Additive.

We now need to adjust the settings in order to create a life-like flame effect. First of all, change the Scale base property to a small value, around 0.2.

Now, to give the particles the upwards motion, we need to set the Initial Angle and the Initial Velocity, which will cause the particles to move at the given speed at the given angle. 270 for the angle and 100 for the velocity should be sufficient.

Note: you can visually adjust the angle by using the dial control to the left of the angle text.

In order to make the particles look organic, we need to add random behaviors to the particles, and this is done with the Offset options. The offset number box will specify values above and below the initial value and randomly choose a value between the new maximum and minimum.

A practical example will best demonstrate this. Set the Initial Angle Offset to 15. This will then mean that the Initial Angle could be any value between 255 and 285. In doing this you will see the fire appear to spread out.

A problem now is that all the flames have a fixed end-point.  This looks very inorganic. In order to change this, the Life Time setting comes into play. The default base value of 30 is probably okay for this example, but in order to add randomness, we again adjust the Offset property. A value of around 8 gives a nice natural flicker to the flames.

Now I will introduce the concept of Increase properties. This value, as the name suggests, increases or decreases the base value of the property as it undergoes its lifetime. Again, a practical example is probably best to show this. To make the fire look more convincing, we want to make the flames fade out and shrink as they get further from the source (or they get older).

 In order to do this, we set a negative value of increase for the Scale property. About -2.00 produces a good effect.

We can do the same for the Opacity value, as well as setting an offset for it, a base of 255 with an offset of 100 and an increase of -1.5 looks good.

You should now have something that looks similar to this:

As a base for flames, this is not bad, but we can further improve it by adjusting the Max Particles and the Particles to Emit.

Note: These values have a direct impact on the performance of your game, the more particles; the more processing they require; the slower your game will run. Monitor the FPS and Particles count in the Particle Viewer.

Setting the Max Particles to 100 and the Particles to Emit to 3 seems to give the fire the thickness it requires.

Some further additions: Adjust the settings to however you see fit, but here are some example additional values: Scale Offset: 0.05, Rotation Offset: 180.

Example final settings:

Now the flames have been created and you understand how the different values work in relation to how they affect the particles, we shall add the smoke.

In the Emitters pane, Add a new emitter and rename it to Smoke.

Like before, click Select Images and this time choose “002-Smoke”. You will see the smoke then appears in front of the flames, so we have to go to the Emitters pane, and while the Smoke emitter is selected, click on the Up arrow .

We have to now apply similar settings to the smoke, and as you know how the majority of the parameters now work some of this will be a brief overview.

Set the scale to 0.2, the Max Particles to 200, the Initial Angle to 270, the Initial Velocity to 120, and the Life Time to 55. This then sets up the base for the smoke.

You will notice now how the smoke now slightly overlaps the flames, which is not what we want, and we can account for this by adjusting the Start Point for the emitter.

Important: The start points for the Emitters are all relative to the central position of the particle system (where the two guide lines meet in the center).

When adjusting the position: Note the relative position which is shown in the coordinates at the bottom of the Particle Viewer.

 

Click Select Start Point. You will see it becomes selected.

You can click it again to deselect it.

Now, when you move your mouse into the Particle Viewer area, you will see the Start Point of the emitter follows your mouse. Look at the coordinates and wait until it becomes (-5, 15) which is a value that works well with this image. Click at the desired location to set it. This will deselect the select start point button and commit the change.
