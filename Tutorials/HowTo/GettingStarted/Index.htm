Getting Started

Welcome to Express Game Maker!

This tutorial explorer can help you learn how to use EGM fast and easy.

First, let's learn the basics starting with the user interface. The user interface is easily customizable by docking, auto-hiding, and clicking on the top borders and dragging. Let's try it out:

  1. To auto-hide the Tiles explorer (docked at the left), click .
  2. Now, place your mouse on the vertical Tiles tab and it will auto-show. Useful for less used explorers.
  3. Now, dock it back by clicking .
  4. Try and place the explorer somewhere else by clicking the border and dragging it around and dropping it on .
  5. Remember, you don't always have to dock. You can also allow them to float as a separate window.
